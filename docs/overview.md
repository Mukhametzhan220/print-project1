# Обзор проекта

## Что это

**Printo KZ** — мобильное веб-приложение для заказа печати документов. Пользователь загружает файл, настраивает параметры печати и оплачивает заказ — всё в рамках пошагового интерфейса из 8 экранов.

## Цели проекта

- Предоставить удобный мобильный интерфейс для заказа печати
- Реализовать полный пользовательский флоу: от авторизации до оплаты
- Продемонстрировать mobile-first подход к разработке на Next.js

## Целевая аудитория

Пользователи в Казахстане, которым нужно распечатать документы. Интерфейс поддерживает два языка (EN/RU) и локальные способы оплаты (Kaspi QR).

## Текущий статус

Проект — функциональный **прототип**. Есть frontend (8-шаговый флоу на Next.js) и backend (REST API на FastAPI с заглушками SMS/платежей). **Frontend и backend пока не интегрированы** — весь стейт фронта живёт в React Context и теряется при перезагрузке страницы. Backend работает автономно и доступен через Swagger UI.

## Технологический стек (frontend)

| Технология | Версия | Назначение |
|-----------|--------|------------|
| Next.js | 15.3.2 | Фреймворк (App Router) |
| React | 19.1.0 | UI-библиотека |
| TypeScript | 5.8.3 | Типизация |
| CSS (vanilla) | — | Стилизация без внешних библиотек |
| ESLint | 9.25.1 | Линтинг |

Внешние UI-библиотеки не используются. Весь интерфейс построен на кастомном CSS с CSS-переменными.

Стек backend (FastAPI + PostgreSQL + Redis + MinIO) описан в [../backend/README.md](../backend/README.md).
