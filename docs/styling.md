# Стилизация

## Подход

Проект использует **vanilla CSS** без внешних библиотек (Tailwind, styled-components и т.д.). Все стили определены в одном файле `frontend/app/globals.css` и используют CSS-переменные для дизайн-токенов.

## Дизайн-токены (CSS-переменные)

```css
:root {
  --bg: #f4f6fb;         /* Фон приложения */
  --surface: #ffffff;     /* Фон карточек и компонентов */
  --text: #131722;        /* Основной цвет текста */
  --muted: #697488;       /* Вторичный текст, подписи */
  --primary: #295cff;     /* Акцентный синий (кнопки, ссылки) */
  --stroke: #e5e9f3;      /* Границы и разделители */
}
```

## Фон страницы

```css
background: radial-gradient(circle at top, #eef3ff, var(--bg) 45%);
```

Радиальный градиент от светло-голубого к основному фону. Создаёт мягкий визуальный акцент в верхней части экрана.

## Основные CSS-классы

### Layout

| Класс | Описание |
|-------|----------|
| `.screen` | Контейнер страницы. Max-width: 420px, min-height: 100vh, вертикальный flex |
| `.screen__header` | Блок заголовка страницы |
| `.screen__content` | Блок контента с gap: 14px |

### Кнопки

| Класс | Описание |
|-------|----------|
| `.btn` | Базовые стили кнопки (border-radius: 14px, padding: 14px) |
| `.btn--primary` | Синий фон, белый текст |
| `.btn--full` | Ширина 100% |
| `.btn--primary:disabled` | Полупрозрачность (opacity: 0.5) |

### Формы

| Класс | Описание |
|-------|----------|
| `.field` | Обёртка поля (label + input), flex column с gap: 8px |
| `.field input` | Стилизованный инпут (border-radius: 14px) |
| `.field select` | Стилизованный селект |
| `.toggle-row` | Строка с переключателем (flex, space-between) |

### Карточки

| Класс | Описание |
|-------|----------|
| `.app-card` | Карточка меню (dashboard) |
| `.payment-card` | Карточка способа оплаты |
| `.payment-card.active` | Активный способ — синяя граница + shadow |
| `.selected-file` | Карточка выбранного файла |
| `.upload-box` | Зона загрузки файла |
| `.pdf-preview` | Контейнер предпросмотра PDF |
| `.payment-total` | Блок итоговой суммы |

Все карточки имеют общие стили:
```css
background: var(--surface);
border: 1px solid var(--stroke);
border-radius: 16px;
padding: 14px;
```

### Навигация

| Класс | Описание |
|-------|----------|
| `.bottom-nav` | Нижняя панель навигации (grid, 5 колонок) |
| `.bottom-nav a` | Элемент навигации |
| `.bottom-nav a.active` | Активный элемент — синий текст, голубой фон |

### Прогресс

| Класс | Описание |
|-------|----------|
| `.progress-wrap` | Контейнер прогресс-бара |
| `.progress-meta` | Метаданные (текст «Step X of Y») |
| `.progress-bar` | Фон прогресс-бара (серый) |
| `.progress-bar span` | Заполненная часть (градиент синий→голубой) |

### Утилиты

| Класс | Описание |
|-------|----------|
| `.hint` | Подсказка (мелкий серый текст) |
| `.hint a` | Ссылка в подсказке (синяя) |
| `.lang-switch` | Переключатель языка (pill-кнопки) |
| `.lang-switch button.active` | Активный язык — белый фон |
| `.card-grid` | Сетка карточек (grid, gap: 10px) |

## Типографика

- **Шрифт:** Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- **Заголовок страницы:** 1.5rem, bold
- **Обычный текст:** 1rem
- **Подписи:** 0.95rem, цвет `--muted`
- **Мелкий текст:** 0.85–0.88rem

## Адаптивность

Приложение спроектировано по принципу **mobile-first**:

- Контент ограничен шириной **420px** (`width: min(100%, 420px)`)
- На широких экранах контент центрируется (`margin: 0 auto`)
- Нижний padding 88px учитывает фиксированную нижнюю навигацию
- Нет медиа-запросов — интерфейс всегда мобильный
